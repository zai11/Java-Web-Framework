Introduction:

Java Web Framework provides an easy way to develop web pages using the Java programming language. JWF provides various functions and classes to easily interface between Java and HTML. As seen in the following code, a new page is created; the title of the page is set to “Home”; and a paragraph is add that says “This is a paragraph”.

Page page = new Page();

pages.add(page);

page.setTitle(“Home”);

page.add(new Paragraph(“This is a paragraph”));

The equivalent HTML would be:

<!DOCTYPE html>

<html>

  <head>

    <title>Home</title>

  </head>

  <body>

    <p>This is a paragraph</p>

  </body>

</html>
